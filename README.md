## :star2: About the Project
<pre>📔 Section 12-Courage: Digital Interactive Scrapbook A bespoke, high-performance digital yearbook created for the class of Section 12-Courage (S/Y 2025–2026).
  This project transforms the traditional static class website into a tactile, nostalgic "scrapbook" experience.
  🌟 Overview
  The 12-Courage Digital Scrapbook is designed to preserve school memories through a "Digital Tactility" philosophy. It mimics the aesthetic of a physical journal—complete with washi tape, polaroid frames, and organic, rotated layouts—while utilizing modern web technologies for a seamless user experience. 🛠️ Key Features 1. Dynamic Media Lightbox A versatile, multi-purpose modal system that intelligently detects content: Images: Automatically scales and fits high-res pictorials using object-fit: contain. Videos: Supports MP4 playback for event highlights (e.g., Christmas Party). Emojis: Lightweight placeholders for quick storytelling. 2. "Complete Size" Responsive Grid A custom CSS Grid implementation that ensures: Desktop: An artistic, overlapping collage layout. Mobile: Automatic transformation to a single-column, "complete size" view for maximum readability on small screens. 3. Smart Image Fitting Across the entire site, images are governed by a "Frame-First" logic: Thumbnails: Use object-fit: cover to ensure frames are always filled and neat. Full View: Use object-fit: contain to ensure no part of a memory is cropped out when viewed closely. 4. Intersection Observer Animations To maintain performance and visual mystery, student cards and memory blocks use a scroll-triggered fade-in effect. 🚀 Technical Stack Technology Use Case HTML5 Semantic structure for memories, videos, and student profiles. CSS3 Custom variables (theming), Flexbox/Grid, and object-fit logic. JavaScript DOM Manipulation, media detection logic, and scroll animations. Google Fonts Caveat (handwritten), Playfair Display (elegant serif), and DM Sans. 📂 File Structure Plaintext ├── index.html # Main structure & layout ├── style.css # Scrapbook aesthetics & responsive breakpoints ├── script.js # Lightbox logic & scroll observers ├── assets/ # Photos (e.g., pictorial.jpg) └── videos/ # MP4 files (e.g., christmasparty.mp4) 🔧 Installation & Setup Clone the Repository: Bash git clone https://github.com/your-username/12-courage-scrapbook.git Add Media: Place your class photos in the /assets folder and videos in the /videos folder. Launch: Open index.html in any modern web browser. No build steps or dependencies required. 👨‍💻 Maintainer Developed for Section 12-Courage (S/Y 2025–2026) Focusing on preserving memories, one frame at a time.</pre>
