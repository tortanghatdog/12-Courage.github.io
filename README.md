# 12-Courage.github.io

<div align='center'>

<p> Gemini said This project is a Digital Interactive Scrapbook, custom-designed as a class yearbook for Section 12-Courage (S/Y 2025–2026). Unlike a traditional static website, this project blends a nostalgic, "hand-made" physical aesthetic with modern web functionality. It uses a layered, collage-based UI to capture school memories in a way that feels personal and organic. 🎨 Design Philosophy: "Digital Tactility" The core concept is to mimic a physical journal. This is achieved through: Washi Tape & Paper Textures: UI elements are "taped" onto the screen using semi-transparent CSS overlays. Organic Layouts: Photos and cards are slightly rotated and offset (using CSS transforms) to avoid the rigid look of standard websites. Dynamic Lighting: A custom-built Lightbox simulates picking up a physical photo to look at it closely, blurring the background to focus on the memory. 🛠️ Technical Features Adaptive Media Lightbox: A versatile modal system that dynamically detects and displays three types of content: High-Resolution Images: For class pictorials and events. Video Embeds: For event highlights like the Christmas Party. Thematic Emojis: For quick, lightweight visual storytelling. Intersection Observer Animations: To prevent visual clutter, student cards and memories "fade-in" only as the user scrolls down, creating a guided narrative experience. Mobile-First Responsiveness: A custom grid system that automatically snaps collage cards to "Complete Size" (full-width) on mobile devices to ensure readability on small screens. Smart Media Fitting: Uses object-fit: cover logic to ensure that photos and video thumbnails always fill their frames perfectly without distortion, regardless of the original file's dimensions. 📂 Project Structure index.html: The semantic structure of the yearbook, organized into logical sections: Cover, Photos, Videos, People, and Memories. style.css: The visual engine, containing the scrapbook theme, custom fonts (Caveat and Playfair Display), and the responsive grid system. script.js: The functional layer handling navigation, the media lightbox logic, and scroll-triggered animations. 🎯 Target Audience Primarily designed for the students and faculty of Section 12-Courage. It serves as a permanent, accessible digital archive that preserves the "vibe" of their graduating year in a format that is much more engaging than a simple photo folder.</p>
